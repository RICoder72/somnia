### Container Management

```bash
# Rebuild after code changes
docker stop somnia && docker rm somnia
docker build -f /data/somnia/docker/Dockerfile -t somnia:latest /data/somnia
docker run -d --name somnia --restart unless-stopped \
  -p 8010:8010 \
  -v /volume1/docker/super-claude/somnia/db:/data/somnia/db \
  -v /volume1/docker/super-claude/somnia/logs:/data/somnia/logs \
  -v /volume1/docker/super-claude/somnia:/app:ro \
  -e SOMNIA_PORT=8010 -e SOMNIA_DEBUG=true \
  -e OP_SERVICE_ACCOUNT_TOKEN="$OP_SERVICE_ACCOUNT_TOKEN" \
  --network super-claude_super-claude-net \
  somnia:latest

# Quick restart (code mounted read-only, so changes apply)
docker restart somnia

# View logs
docker logs -f somnia
```

### API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/` | GET | Health check, shows auth status |
| `/status` | GET | Graph stats, dream readiness |
| `/consolidate` | POST | Trigger dream (`force`, `dry_run` options) |
| `/inbox` | GET/POST | List/add inbox items |
| `/nodes` | GET/POST | List/create nodes |
| `/nodes/<id>` | GET | Get node with edges |
| `/edges` | POST | Create edge |
| `/dreams` | GET | List dream sessions |
| `/dreams/<id>` | GET | Get dream details |
| `/search?q=` | GET | Full-text search |
| `/init` | POST | Reinitialize database |
