# Code of Conduct

## Standard

Be respectful. Be constructive. Assume good faith.

This project is built by people who value honest, challenging intellectual
engagement. Disagreement is welcome — that's how good systems get better.
What's not welcome is personal attacks, harassment, or bad-faith behavior.

## Scope

This applies to all project spaces: issues, pull requests, discussions,
and any other communication channel associated with the project.

## Enforcement

Unacceptable behavior can be reported to the project maintainer at
matt@burrillvilleri.gov. Reports will be reviewed and responded to
as appropriate. The maintainer reserves the right to remove, edit,
or reject contributions that violate this code of conduct.
